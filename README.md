A simple Web-Chat application in C 
<br><br>
After compiling<br>
Run the server with 
  ./server [port]
  <br>
  Default server IP is localhost
 <br><br>
Run the client with
  ./client [ip] [port]
  <br>
  Where [ip] and [port] the IP and Port of the server

Available commands for the client are:
<br>
<ul>
<li>
/creategroup [groupname]
</li>
<li>
/deletegroup [groupname]
</li>
<li>
/join [groupname]
</li>
<li>
/msg [message]
</li>
<li>
/smsg [groupname] [message]
</li>
<li>
/add [username]
</li>
<li>
/remove [username]
</li>
<li>
/show friends
</li>
<li>
/pm [username> [message]
</li>
</ul>
